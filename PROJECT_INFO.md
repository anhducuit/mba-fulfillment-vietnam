## Development Workflow
- **Push Direct to GitHub**: Updates and edits should be pushed directly to the GitHub repository.
- **Vercel Auto-deploy**: The project is configured to auto-deploy to Vercel upon pushing to the `main` branch.
- **Local Testing**: While pushing direct is preferred, local testing can be done using `npm run dev` at [http://localhost:8080/](http://localhost:8080/).
