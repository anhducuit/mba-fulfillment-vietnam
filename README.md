# MBA Fulfillment Vietnam

Dự án Website MBA Fulfillment Việt Nam.

## Công nghệ sử dụng

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Hướng dẫn cài đặt

```sh
npm install
npm run dev
```

## Tính năng

- Quản lý đơn hàng (Order Management System)
- Kết nối đa kênh (Shopee, Lazada, TikTok Shop)
- Tra cứu vận đơn
